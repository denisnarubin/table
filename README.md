# DataTable Component

Управление табличными данными с полным CRUD-функционалом, реализованное на React с использованием Ant Design.

## 🚀 Функциональные возможности

### Основные функции
- **CRUD операции**: Создание, чтение, обновление и удаление записей
- **Поиск**: Поиск по всем полям таблицы (имя, дата, значение)
- **Сортировка**: Сортировка по всем колонкам с учетом типа данных
- **Модальное окно**: Добавление и редактирование записей через модальное окно
- **Валидация**: Встроенная валидация данных при добавлении/редактировании

### Интерфейс
- **Колонки**: Имя, Дата, Значение, Действия
- **Кнопки действий**: Редактировать (иконка карандаша) и Удалить (иконка корзины)
- **Поисковая строка**: Расположена над таблицей для быстрого поиска
- **Кнопка "Добавить"**: Для создания новых записей

## 🛠 Технологический стек

- **React 19.2.4** - JavaScript библиотека для создания пользовательских интерфейсов
- **Vite 8.0.1** - Современный сборщик и сервер разработки
- **Ant Design 6.3.3** - UI-библиотека с готовыми компонентами

## 🎨 Особенности реализации

### Валидация данных
- **Имя**: Минимум 2 символа, максимум 100 символов
- **Дата**: Обязательное поле
- **Значение**: Только положительные числа

### Локализация
Полностью русифицированный интерфейс с использованием локали 'ru' в Ant Design и Day.js.

### Поиск
Реализован поиск по всем полям таблицы с автоматическим фильтрованием данных в реальном времени.

### Сортировка
- По имени: Лексикографическая сортировка с учетом русского алфавита
- По дате: Хронологическая сортировка
- По значению: Числовая сортировка

### Модальное окно
- Автоматическое переключение между режимами "Добавить" и "Редактировать"
- Сохранение состояния формы при закрытии
- Валидация перед сохранением

## 📱 Адаптивность

Проект поддерживает различные размеры экранов:
- Полная функциональность на мобильных устройствах
- Адаптивная верстка с учетом маленьких экранов
